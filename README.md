# Mergkol

Merging multiple kotlin source code files into single file, removing all package declaration.

## Installation

`go install github.com/CombatMage/go-mergkol`

## Usage

`go-mergkol.exe -h`

`go-mergkol.exe -dir input -o merged.kt`
